# Knowledge Base RAG System with Transformers

A complete Retrieval-Augmented Generation system that processes PDF documents, chunks them, generates embeddings using transformer models, and stores them in ChromaDB for efficient retrieval.

## Features

- PDF document loading and processing
- Intelligent text chunking with overlap
- Transformer-based embeddings (Sentence Transformers)
- ChromaDB vector database storage
- Configurable chunk sizes and overlap
- Support for multiple embedding models
